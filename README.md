## Getting Started

This is a [Next.js](https://nextjs.org) project using Node.js >= 22.

Install the dependencies:
```bash
yarn install
```

And run the development server:

```bash
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
