# Security Policy

## Found a Security Issue?

If you discover a security problem with this website, please let me know privately so I can fix it before it becomes a bigger issue.

**📧 Email me at:** [admin@appcheckwizard.com]  
**I'll respond within:** 5 business days

### What I Need from You
- A description of what you found
- How you found it (so I can reproduce it)
- What you think could go wrong

### What Happens Next
- **If it's a real issue:** I'll fix it ASAP, update the site, and send you a public thank you (if you're okay with it).
- **If it's not a security risk:** I'll explain why and defer fixing.

## Repository Security Practices

- It automatically scans for vulnerable code dependencies
- All code changes require review before going live

***
**Important:** Please don't post security issues publicly on GitHub. Email me privately instead so I can fix the problem safely.

Thank you for your time and effort in helping me improve my project!
